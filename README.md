# おぼえてて (remember_me)

子供との大切な思い出を記録・共有するデジタルアプリ

## 📱 アプリ概要

「おぼえてて」は、子供の成長過程で生まれる貴重な瞬間をデジタルで記録し、家族で共有できる思い出アプリです。子供の可愛い一言や日常の小さな出来事、成長の記録を写真とテキストで残し、いつでも振り返ることができます。

### 🎯 コンセプト
- **思い出のデジタル化**: 子供の成長記録を失くすことなく、永続的に保存
- **家族での共有**: 家族みんなで思い出を共有できる

## ✨ 主要機能

- 📝 **思い出の記録**: テキストや写真で日常の出来事を記録
- 💬 **名言・エピソード保存**: 子供の可愛い一言や面白いエピソードを保存
- 📅 **タイムライン表示**: 日付順に思い出を整理・表示
- 👨‍👩‍👧‍👦 **家族共有**: 家族メンバー間で思い出を共有
