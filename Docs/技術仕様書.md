## CI/CD

GitHub Actions で自動実行：
1. コード品質チェック (ESLint, Prettier, RuboCop)
2. テスト実行 (Vitest, RSpec)
3. セキュリティチェック (Brakeman)
4. デプロイ

## 開発環境

- Node.js 20.x LTS
- Ruby 3.3.0
- PostgreSQL 16
- Docker & Docker Compose
