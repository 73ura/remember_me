# タイムライン表示
目的：保護者が／タイムラインにアクセスしたとき／記録を素早く振り返るために／投稿を新しい順で連続閲覧できる。
成功指標：API→p95 応答 <= 300ms / UI→初期描画 <= 1.0s、1ページ10件、空は「投稿はまだありません」

受入基準：
- 正常）Given DBにこのユーザーの投稿が1件以上ある、When /timeline、Then 最大10件がcreated_ad descで表示され、ページングUI（次へ）が出る
- 空）Given 0件、When /timeline、Then 200 + items: []、空文言「投稿はまだありません」
- 異常（未ログイン））Given 未ログイン、When /timeline、Then APIは401 + { errors:[{code:"auth.unauthorized"}] }、UI：ログインページに遷移

API骨子：
- GET //timeline?child_id=<id>&page=<int>&per_page=<int>
	200
	```json
	{
  "items": [
    {
      "id": 123,
      "child_id": 10,
      "author_id": 5,
      "text": "今日の名言…",
      "photos": ["https://.../p1.jpg"],   // 0～5枚
      "created_at": "2025-09-01T12:34:56Z",
      "like_count": 0,
      "comment_count": 0,
      "child_color": "#A3D1FF",
      "child_thumbnail_url": "https://.../thumb.jpg"
    }
  ],
  "page": 1,
  "per_page": 10,
  "total_pages": 3
}
	```
（空可）
	
# 投稿
目的：保護者が／投稿ページにアクセスしたとき／思い出を素早く残すために／日付・子供・本文・写真を入力して送信できる。
成功指標：成功トースト「投稿しました」+タイムラインへ、失敗は最初のエラーへフォーカス、送信中はボタンを無効化し「送信中・・・」を表示

受入基準：
- 正常）Given child_id 選択・text が 200字以内・photos <= 5、When 送信、Then 201 + 作成post返却、成功トースト表示、タイムライン先頭に反映（またはリダイレクト）
- 異常（バリデーション））Given text 空（or 201字 / 写真6枚）、When 送信、Then 422 + errors[{code,field,message}]、エラー要約と最初のエラーにフォーカス
- 境界）Given textちょうど200字、When 送信、Then 成功

API骨子：
- POST /posts（Content-Type: application/json か multipart/form-data）
	Request
```json
{
  "child_id": 10,
  "text": "…",         // <=200
  "photos": ["file..."],// 0..5
  "recorded_at": "2025-09-01T12:34:56Z", // 任意
  "visibility": "private",
  "category": "quote"   // 任意
}
```
201（推奨）
```json
{ "post": { "id": 123, "...": "..." } }
```
422
```json
{ "errors": [{ "code": "validation.blank", "field": "text", "message": "テキストを入力してください" }] }
```

エラーポリシー（UI×HTTP）：
- タイムライン：
	- 400（クエリ不正）→エラーバナー+再読み込み
	- 401（未ログイン）→API:401、UIは/loginへ遷移
	- 403/404（権限/不存在）→文言「アクセス権がありません」 or 子ども選択へ戻る
	- 500/503（障害/一時停止）→ リトライ と問い合わせへの導線
- 投稿：
	- 422（入力不正）→フィールド別エラー+先頭へフォーカス
	- 401/403→保存しない+ログイン/権限案内
	- 429（連続投稿）→Retry-After 秒を表示（30秒）
	- 501（内部）/503（メンテ）→ドラフト維持+リトライ

個人情報・安全配慮：
- 公開範囲：private（初期はこれのみ）
- 画像：EXIF保存しない／1枚<= 5MB／合計<=20MB（仮）
- 名前表記：ニックネーム（設定で将来切替可
